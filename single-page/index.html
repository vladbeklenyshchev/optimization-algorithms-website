Описание проекта

В проекте реализовано

Шесть алгоритмов для решения методами первого порядка

  • метод градиентного спуска с постоянным шагом;
  • метод найскорейшего градиентного спуска;
  • метод покоординатного спуска;
  • метод Гаусса-Зейделя;
  • метод Флетчера-Ривса;
  • метод Дэвидона-Флетчера-Пауэлла;

Три алгоритма для решения методами второго порядка

  • метод Ньютона;
  • метод Ньютона-Рафсона;
  • метод Марквардта;

При реализации использовался такой источник: А. В. Пантелеев, Т. А. Летова "Методы опитимизаций в примерах и задачах", 2005г.

Использованные технологии и фреймворки

  • JS для реализации методов;
  • Bootstrap для простоты создания адаптивного дизайна;
  • библиотеку jQuery для создания визуальных эффектов;
  • плагин на jQuery maskedinput для создания маски для ввода функции;
  • Katex для рендеринга математических формул;
  • Gulp для сборки проекта;
  • в качестве системы контроля версий использовалась Git система.
